# TAD5142 Module

This module board is a headphone output module featuring TI’s stereo audio DAC TAD5142. It supports sample rates up to 192 kHz and can add a headphone output to your microcontroller board or other hardware via an I2S connection.

## How to buy
- Tindie(International)  
- BASE(Japan)

## Specifications
- Power: 5V input, onboard 3.3V regulator (AP7361)
- DAC IC: [TAD5142](https://www.ti.com/product/TAD5142)
  - Channels: stereo output
  - Sample Rate: 192kHz(max.)
  - I2S Slave
- Audio Jack: [Green Thonkiconn Stereo 3.5mm Jack Socket](https://www.thonk.co.uk/shop/thonkiconn/)
- Power IC: [AP7361EA-33E-13](https://www.diodes.com/assets/Datasheets/AP7361EA.pdf)
